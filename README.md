# 🏠 House Price Prediction using Machine Learning

**Author:** Mani Sharan Bommakanti  
**GitHub:** [github.com/manisharan77](https://github.com/manisharan77)  
**Tech Stack:** Python, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn

---

## 📌 Project Overview

A complete **Machine Learning regression project** that predicts house prices based on features like location, area, number of rooms, amenities, and more. Includes full EDA, feature engineering, model comparison, and visualization.

---

## 🚀 Features

- ✅ Realistic dataset generation with 500 houses and 10+ features
- ✅ Full Exploratory Data Analysis (EDA) with statistical insights
- ✅ Feature engineering — room density, amenity score, age category
- ✅ Missing value handling using median imputation
- ✅ 5 regression models trained and compared
- ✅ Evaluation using RMSE, MAE, and R² metrics
- ✅ Feature importance analysis
- ✅ Full visualization dashboard
- ✅ Predict prices for new custom houses

---

## 🛠️ Tech Stack

| Tool | Purpose |
|---|---|
| Python | Core programming language |
| Scikit-learn | ML models, preprocessing, evaluation |
| Pandas & NumPy | Data handling and feature engineering |
| Matplotlib & Seaborn | EDA and result visualization |

## ⚙️ Installation & Usage

```bash
# 1. Clone the repository
git clone https://github.com/manisharan77/house-price-prediction.git
cd house-price-prediction

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the project
python house_price_prediction.py
```
